Dataset 1:

Dataset Name: Audible Complete Catalog

File Name(s): google_books_1299.csv, google_books_dataset.csv

Source: Kaggle

Relevant fields: Title, Authors, Language, Rating (Out of 5), Voters, Price, Page Count, Publisher

** Link: https://www.kaggle.com/datasets/bilalyussef/google-books-dataset **
